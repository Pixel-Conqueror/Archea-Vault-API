# Archea-Vault-API
## Installation

**Clone the repository:**
```
git clone https://github.com/Pixel-Conqueror/Archea-Vault-API.git
```

**Enter directory:**
```
cd Archea-Vault-API
```

**Install dependencies:**
```
npm install
```

**Create .env:**
```
cp .env.exemple .env
```

**Create container:**
```
docker-compose up -d
```
